Cameras
=======

DaoAI Vision Studio support multiple types of cameras. 

* DaoAI Camera: This is our high precision structured light camera, the precision of this camera can be up to 0.02mm.
* DaoAI Mini Camera: This is a cost-effective and light camera, which usually have the precision for few mm. 
* DaoAI Laser Camera: This is a laser profile camera, which is super suitable for moving objects recognition on conveyor.
* DaoAI Gige & USB3 Camera: This is a industrial 2D camera. We provide multiple different models.
